Twitch Streamer For Linux
=========================

This is a script dedicated to stream to Twitch.tv. it also has Webcam support!

Dependencies:
-------------

        avconv pulseaudio alsa xwininfo pactl ffmpeg

###Suggested:

        pavucontrol
